# Trumbowyg Meteor

WYSIWYG editor in Meteor

Original project : https://github.com/Alex-D/Trumbowyg

## Getting Started

Just call 

```
$('textarea').trumbowyg();

```

### Installing

```
meteor add jessyleite:meteor-trumbowy

```

##Version

Trumbowyg v2.7.0
